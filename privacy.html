Privacy Policy

Last updated: April 11, 2026

Pennivo is built on a simple principle: your writing belongs to you. We collect as little data as technically possible, and in most cases, none at all.

Desktop App (Windows)

The Pennivo desktop app runs entirely on your local machine. There are no accounts, no analytics, no telemetry, and no crash reporting.

  • No data collection. The app does not send any personal data, usage data, or file contents to any server.
  • All files stay local. Your documents are stored as plain .md files on your own disk. Pennivo never uploads, syncs, or transmits them.
  • No account required. There is no sign-up, no login, and no user profile.
  • No analytics or telemetry. The app does not track how you use it, what you write, or how often you open it.
  • Auto-update checks. The only network request the app makes is checking the GitHub Releases API for new versions. This request contains no personal information.

Android App

The Android version of Pennivo follows the same offline-first approach as the desktop app.

  • No data sent to any server. All editing happens on-device. No file contents, metadata, or usage information leaves your phone or tablet.
  • Local storage only. Files are stored in the app-specific directory on your device. Settings are stored locally using Capacitor Preferences.
  • No third-party SDKs. Pennivo does not include any analytics, advertising, or data-collection SDKs.
  • Permissions. The app may request camera and storage permissions solely for inserting images into your documents. These permissions are never used for any other purpose.
  • No account required. Same as desktop — no sign-up, no login, no profile.

Website (pennivo.app)

The Pennivo website is a static site hosted on Cloudflare Pages.

  • Cloudflare Web Analytics. We use Cloudflare Web Analytics, a privacy-focused analytics service that does not use cookies, does not collect personally identifiable information, and does not track visitors across sites.
  • No user accounts or forms. The website does not collect any personal information. There are no forms, no sign-ups, and no comment sections.
  • Standard server logs. As with any website, Cloudflare’s CDN may log IP addresses and request metadata in standard server logs. This is standard web hosting infrastructure and is governed by Cloudflare’s own privacy policy.
  • Theme preference. Your light/dark theme choice is saved in your browser’s localStorage. This data never leaves your browser.

Cookies

Pennivo does not set any cookies. Cloudflare may set standard security cookies (such as __cfruid) as part of its CDN and DDoS protection services. These are strictly functional and are not used for tracking or advertising.

Third-Party Data Sharing

We do not share, sell, rent, or trade any user data with third parties. There is no advertising in any Pennivo product.

Open Source

Pennivo is open-source software licensed under the MIT License. You can inspect the full source code on GitHub to verify every claim in this policy.

Changes to This Policy

If this policy is ever updated, the changes will be posted on this page with an updated date. Because Pennivo collects essentially no data, meaningful changes are unlikely.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub issue tracker.
